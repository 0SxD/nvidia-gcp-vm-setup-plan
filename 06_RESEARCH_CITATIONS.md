## Verification command (bulk)

To verify all URLs return 200/301:

```bash
for url in $(grep -oE 'https?://[^ )]+' nvidia_gcp_vm_setup_plan/*.md | awk -F: '{$1=""; print}' | sort -u); do
  printf "%-80s %s\n" "$(echo "$url" | head -c 80)" "$(curl -sI -o /dev/null -w '%{http_code}' "$url")"
done | tee research_notes/url_verify_$(date +%F).log
```
